# 截图目录

本目录用于存放项目的截图文件。

## 所需截图

请准备以下截图并放入此目录：

### APP 界面截图

| 文件名 | 说明 |
|--------|------|
| app_home.png | APP 主页界面 |
| app_scripts.png | 脚本列表界面 |
| app_logs.png | 执行日志界面 |
| app_profile.png | 个人中心界面 |
| app_login.png | 登录界面 |
| app_register.png | 注册界面 |

### 后台管理截图

| 文件名 | 说明 |
|--------|------|
| admin_dashboard.png | 后台仪表盘 |
| admin_users.png | 用户管理页面 |
| admin_scripts.png | 脚本管理页面 |
| admin_records.png | 执行记录页面 |
| admin_login.png | 后台登录页面 |

## 截图要求

- **分辨率**: 建议 1080x1920 (手机) 或 1920x1080 (网页)
- **格式**: PNG 格式
- **内容**: 请隐藏敏感信息（如真实用户名、密码等）

## 添加截图

1. 准备截图文件
2. 按照上述命名规范命名
3. 放入此目录
4. 更新 README.md 中的截图引用
